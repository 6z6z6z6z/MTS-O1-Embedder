计算资源违规占用管理条例
由于服务器资源紧缺，为了保证服务器的正常使用，服务器限额全局生效。
- 违规占用计算资源
  - 每台机器使用CPU核数超过1/4
  - 每台机器内存使用超过1/2
  - 显卡占用限制
    - （（V100/A100）2块 （3090）2块（4090）2块）4块
    - 即：V100/A100可以同时使用2块，3090、4090可以同时使用2块，上述显卡共可以同时使用4块
    - pangpang号可以占用4块1080显卡
- 异常进程未清理（ps ux命令STAT含有T或Z的进程）
- 显卡异常占用，如显存占满但核心利用率0%
- IO异常占用，IO占用112
