# MTS-O1-Embedder 技术框架图

> 基于 v18（当前最优）配置，NATOPS 数据集（C=24, T=51, 6 类）

---

## 一、系统总览

```

│                          MTS-O1-Embedder 系统架构                               │
```

## 四、对比损失计算

```
                query_emb [18, 1024]          gallery_emb [18, 1024]
                (有梯度, view1)               (detach, view2, EMA参数)
                         │                            │
                         ▼                            ▼
                    L2 normalize                  L2 normalize
                         │                            │
                         │    ┌─── Memory Bank ──┐    │
                         │    │ [90, 1024] 历史   │    │
                         │    │ gallery embeddings │    │
                         │    │ (warmup后启用)     │    │
                         │    └────────┬─────────┘    │
                         │             │              │
                         ▼             ▼              ▼
                    ┌────────────────────────────────────┐
                    │   Concatenate Gallery               │
                    │   all_gallery = [gallery; bank]     │
                    │   → [18+90, 1024] = [108, 1024]    │
                    │   all_labels = [18 + 90] labels     │
                    └───────────────┬────────────────────┘
                                    │
                         ┌──────────┴──────────┐
                         ▼                      ▼
              ┌──────────────────┐   ┌──────────────────────┐
              │ Similarity Matrix │   │ SoftCLT Weights       │
              │                   │   │                       │
              │ sim = q @ g.T / τ │   │ ts_query [18,24,51]   │
              │ τ = exp(log_τ)    │   │ ts_gallery [108,24,51]│
              │ log_τ: learnable  │   │                       │
              │ clamp [-4.6, 0]   │   │ flatten → [18, 1224]  │
              │                   │   │ L2 normalize           │
              │ → [18, 108]       │   │ Euclidean dist         │
              │                   │   │ → [18, 108]            │
              └────────┬─────────┘   │                       │
                       │              │ 同类: w=exp(-d/σ)      │
                       │              │ 异类: w=0              │
                       │              │ 行归一化 → Σ=1          │
                       │              │ → [18, 108] soft mask  │
                       │              └──────────┬────────────┘
                       │                          │
                       ▼                          ▼
              ┌──────────────────────────────────────────┐
              │         Supervised Contrastive Loss       │
              │         (SupCon Form 2 + SoftCLT)         │
              │                                           │
              │  SoftCLT path:                            │
              │    L = -Σ_i [ Σ_j w_ij * s_ij/τ          │
              │              - log Σ_k exp(s_ik/τ) ]      │
              │                                           │
              │  Hard label path (DDP mode):              │
              │    positive_mask: same-label pairs         │
              │    L = -[log(Σ_p exp) - log|P|            │
              │          - log(Σ_all exp)] / Q            │
              └────────────────┬──────────────────────────┘
                               │
                               ▼
              ┌──────────────────────────────────────────┐
              │         ProtoNCE Loss                     │
              │                                           │
              │  EMA class prototypes:                    │
              │    proto[c] = m*proto[c] + (1-m)*centroid │
              │    m = 0.9                                │
              │                                           │
              │  sim = q_norm @ protos.T / τ              │
              │  → [18, 6]                                │
              │  L = CrossEntropy(sim, class_idx)          │
              └────────────────┬──────────────────────────┘
                               │
                               ▼
                total_loss = 1.0 × CL + 0.1 × PL
```

## 五、检索推理阶段

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                           Inference Pipeline                                 │
│                                                                              │
│  Phase 1: Build Gallery (训练集 180 samples, context-only prompt)             │
│  ┌────────────────────────────────────────────────────────────────────────┐  │
│  │  for each train sample:                                                │  │
│  │    ts [24,51] → TSEncoder → Projector → LLM(ctx prompt) → pool → emb  │  │
│  │                                                                        │  │
│  │  → gallery_embeddings [180, 1024]                                      │  │
│  │  → gallery_ts [180, 24, 51]                                            │  │
│  │  → gallery_labels [180]                                                │  │
│  │                                                                        │  │
│  │  Build cosine index: L2-normalize → vector matrix [180, 1024]          │  │
│  └────────────────────────────────────────────────────────────────────────┘  │
│                                                                              │
│  Phase 2: Embed Queries (测试集 180 samples, context-only prompt)            │
│  ┌────────────────────────────────────────────────────────────────────────┐  │
│  │  for each test sample:                                                 │  │
│  │    ts [24,51] → TSEncoder → Projector → LLM(ctx prompt) → pool → emb  │  │
│  │                                                                        │  │
│  │  → query_embeddings [180, 1024]                                        │  │
│  └────────────────────────────────────────────────────────────────────────┘  │
│                                                                              │
│  Phase 3: Hybrid Search (per query)                                          │
│  ┌────────────────────────────────────────────────────────────────────────┐  │
│  │                                                                        │  │
│  │  ┌─── Semantic (α=0.9) ──┐    ┌─── Structural (1-α=0.1) ──┐          │  │
│  │  │                        │    │                             │          │  │
│  │  │  cosine_sim = q @ G.T  │    │  top-M by semantic score    │          │  │
│  │  │  sem = (cos + 1) / 2   │    │  M = max(k×10, 100)        │          │  │
│  │  │  → [180] scores        │    │                             │          │  │
│  │  │                        │    │  for each candidate:        │          │  │
│  │  │                        │    │    z-norm both TS            │          │  │
│  │  │                        │    │    DTW with Sakoe-Chiba      │          │  │
│  │  │                        │    │    sim = 1/(1+dist)          │          │  │
│  │  │                        │    │  min-max normalize           │          │  │
│  │  │                        │    │  → [180] scores              │          │  │
│  │  └───────────┬────────────┘    └──────────────┬──────────────┘          │  │
│  │              │                                 │                         │  │
│  │              ▼                                 ▼                         │  │
│  │         final_score = 0.9 × semantic + 0.1 × structural                 │  │
│  │                              │                                          │  │
│  │                              ▼                                          │  │
│  │                     Top-K (K=5) neighbors                               │  │
│  │                              │                                          │  │
│  │                              ▼                                          │  │
│  │                   Weighted Vote Classification                          │  │
│  │                   weight = neighbor.score                               │  │
│  │                   → predicted class label                               │  │
│  └────────────────────────────────────────────────────────────────────────┘  │
│                                                                              │
└─────────────────────────────────────────────────────────────────────────────┘
```

```
                                   │
                                   ▼
    ┌──────────────────────────────────────────────────────────────────┐
    │                  Epoch 4 ~ 40: Main Training                     │
    │                                                                   │
    │  每个 Epoch:                                                      │
    │  ┌──────────────────────────────────────────────────────────┐    │
    │  │  for batch in train_loader:                               │    │
    │  │    1. view1, view2 = augment(ts)                          │    │
    │  │    2. query_emb = encode(view1, ctx) ← 有梯度              │    │
    │  │    3. gallery_emb = EMA_encode(view2, ctx) ← 无梯度        │    │
    │  │    4. CL = asymmetric_infonce(q, [g; bank]) ← SoftCLT    │    │
    │  │    5. PL = proto_nce(q, prototypes)                       │    │
    │  │    6. loss = 1.0*CL + 0.1*PL                              │    │
    │  │    7. loss.backward() → clip_grad(1.0) → optimizer.step() │    │
    │  │    8. EMA_update(momentum=0.99)                           │    │
    │  │    9. memory_bank.update(gallery_emb, labels)             │    │
    │  │   10. prototype_update(gallery_emb, labels, m=0.9)        │    │
    │  └──────────────────────────────────────────────────────────┘    │
    │                                                                   │
    │  Epoch 结束:                                                      │
    │  ┌──────────────────────────────────────────────────────────┐    │
    │  │  1. scheduler.step() (cosine annealing)                   │    │
    │  │  2. LOO Retrieval on train set (180 samples)              │    │
    │  │     → accuracy, macro_f1                                  │    │
    │  │  3. Smoothed LOO = mean(last 3 epochs)                    │    │
    │  │  4. if smoothed_LOO >= best:                              │    │
    │  │       save model_best.pt                                  │    │
    │  │       reset patience counter                              │    │
    │  │     else:                                                 │    │
    │  │       patience += 1                                       │    │
    │  │  5. if patience >= 15: early stop                         │    │
    │  │  6. save model_last.pt + training_history.json            │    │
    │  └──────────────────────────────────────────────────────────┘    │
    │                                                                   │
    └───────────────────────────────┬──────────────────────────────────┘
                                    │
                                    ▼
                    ┌──────────────────────────────┐
                    │   Post-Training Evaluation     │
                    │                                │
                    │  1. Reload model_best.pt       │
                    │  2. Gallery = train (180)       │
                    │  3. Queries = test (180)        │
                    │  4. gallery: context-only embed │
                    │  5. query:  context-only embed  │
                    │  6. HybridRetriever.search()    │
                    │     k=5, alpha=0.9, weighted    │
                    │  7. Report: Acc, F1, per-class  │
                    └──────────────────────────────┘
```

```
mts_agent/main.py                  ← 入口，配置加载，训练编排
    ├── config.py                  ← 6 个 dataclass 定义所有参数
    ├── tokenization.py            ← DummyTokenizer (离线模式 fallback)
    │
    ├── data/
    │   ├── loader.py              ← MTSDataset (JSONL→dict, 双视图增广)
    │   │   └── augmentations.py   ← TimeSeriesAugmentor (jitter/scale/shift/crop/dropout)
    │   ├── collator.py            ← MultimodalCollator (pad+tokenize+prompt build)
    │   │   └── prompt_builder.py  ← build_full_prompt / build_retrieval_prompt
    │   │       └── prompt_templates.py  ← 数据集特定模板注册
    │   ├── adapters.py            ← infer_ts_input_dim (自动推断通道数)
    │   └── samplers.py            ← BalancedBatchSampler / MultiDatasetBatchSampler
    │
    ├── models/
    │   ├── mts_embedder.py        ← MTSEmbedder (核心: 组装 encoder+projector+LLM)
    │   │   ├── ts_encoder.py      ← TimeSeriesEncoder / PatchTokenizer / ChannelMixer
    │   │   └── projector.py       ← TimeSeriesProjector (MLP + residual)
    │   └── (LLM via transformers AutoModelForCausalLM + peft LoRA)
    │
    ├── engine/
    │   └── trainer.py             ← MTSTrainer (训练循环, 损失, EMA, 评估)
    │       ├── EmbeddingMemoryBank  ← 循环队列存储 gallery embeddings
    │       └── EpochLossTracker     ← 损失聚合
    │
    └── retrieval/
        ├── hybrid_search.py       ← HybridRetriever (cosine+DTW 混合检索)
        └── evaluate_retrieval.py  ← build_gallery / embed_queries / LOO eval
```
